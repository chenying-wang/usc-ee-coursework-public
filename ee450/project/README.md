# EE-450 Project


## Build
GCC need to support C++17 (ISO/IEC 14882:2017).

    $ make


## Run
Please follow the steps
1. Run healthcenter firstly
2. Then run hospital
3. Lastly run student

### Healthcenter

    $ make run_healthcenter

### Hospital

    $ make run_hospital

### Student

    $ make run_student
